# BlogSite

BlogSite is an editorial-grade publishing platform foundation built with TanStack Start, React 19, and Tailwind CSS. This repo currently delivers the public reading experience, admin shell, and SEO surface scaffolding described in the architectural blueprint.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tests

```bash
npm run test
```

## Notes

Sample content lives in `src/lib/content.ts` and powers the initial public routes while backend services are wired up.
